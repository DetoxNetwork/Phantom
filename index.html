Phantom is a way to disable any extension installed by your administrator.
Use this with extreme caution.
Note: this will not work in the newest versions of Chrome! Check you chrome version on chrome://version. Anything below v106 will work.

How to Disable Extensions:

1. Right click this code and "Copy link address".

2. Bookmark this page then right click it and "Edit" it.

3. Then paste the code you just copied into the "URL" box and name it whatever you want.

4. Go to this page and click the bookmark when you get there. (It will bring you to a 404 page, this is not an error!)
